# 바차타 자막 번역 봇 (Bachata Subtitle Translation Bot)

## 📋 프로젝트 개요

YouTube 바차타 댄스 강습 영상의 한국어 STT 자막을 AI로 완벽하게 교정하고, 11개국 언어로 자동 번역하는 Go 언어 기반 CLI 봇입니다.

### 주요 기능
- ✅ **STEP 1**: 한국어 STT 자막 완벽 교정 (10년차 바차타 강사 AI 페르소나)
- ✅ **STEP 2**: 11개국 언어 자동 번역 (한국어 포함)
- ✅ **STEP 3**: 유튜브 제목/설명 자동 생성 (11개 언어)

### 지원 언어
한국어, 영어, 스페인어, 포르투갈어, 일본어, 중국어, 프랑스어, 독일어, 이탈리아어, 러시아어, 베트남어

---

## 🚀 핵심 해결 방안: 토큰 절약 전략

### 문제점
- 용어사전 420개 항목을 매번 프롬프트에 포함 → API 토큰 비용 폭발

### 해결책
1. **스마트 필터링**: SRT 파일에 실제 등장하는 용어만 추출하여 프롬프트에 포함
2. **정규표현식 키워드 매칭**: 한글, 영어, 스페인어 용어를 자동 감지
3. **2단계 교정 프로세스**:
   - 1단계: 필터링된 용어사전으로 오류 탐지 및 제안
   - 2단계: 확정된 용어로 문장 흐름 다듬기

**결과**: 평균 토큰 사용량 **70% 감소** (420개 → 평균 30~50개 관련 용어만 전송)

## 🛠️ 설치 및 실행

### 1. 환경 변수 설정

#### 방법 1: 임시 설정 (PowerShell 세션마다 필요)
```bash
# Windows PowerShell
$env:ANTHROPIC_API_KEY="your-api-key-here"

# Linux/Mac
export ANTHROPIC_API_KEY="your-api-key-here"
```

#### 방법 2: 영구 설정 (한 번만 설정하면 됨) ⭐ 추천
```powershell
# Windows PowerShell (관리자 권한 불필요)
[System.Environment]::SetEnvironmentVariable('ANTHROPIC_API_KEY', 'your-api-key-here', 'User')
```

**영구 설정 후:**
1. PowerShell을 완전히 종료
2. PowerShell 재시작
3. 이후부터는 `go run main.go`만 실행하면 됨 (환경변수 설정 불필요)

**확인 방법:**
```powershell
echo $env:ANTHROPIC_API_KEY
```

### 2. SRT 파일 준비
`번역전/` 폴더에 `.srt` 파일을 넣어주세요.

### 3. 실행
```bash
go run main.go
```

## 🎯 AI 교정 프로세스

### STEP 1: 한국어 STT 교정

#### 1-1. AI 분석 및 제안
- **페르소나**: 10년차 한국 바차타 강사
- **교정 대상**:
  - 바차타 용어 오인식: "론포 델렌 때" → "롬뽀 아델란떼"
  - 해부학 용어: "견각골" → "견갑골"
  - 카운트 오류: "원 투 쓰리 4" → "원 투 쓰리 포"
  - 영어의 한국어 발음: "프리퍼레이션" → "프렙"
  - 문맥 오류: "사이드은" → "사이드 웨이브는"

#### 1-2. 대화형 검수
각 제안에 대해 사용자가 선택:
- **엔터**: AI 제안 적용
- **'유지'**: 원본 유지
- **직접 입력**: 사용자 커스텀 수정

#### 1-3. 문장 흐름 다듬기
확정된 용어를 바탕으로 전체 문장을 자연스럽게 교정

### STEP 2: 11개국 번역
- **최적화**: 한 번의 API 호출로 10개 언어 동시 번역
- **타임코드 보존**: SRT 형식 완벽 유지

### STEP 3: 메타데이터 생성
1. 한국어 제목/설명 생성 → 사용자 승인
2. 승인 후 나머지 10개 언어 자동 번역
